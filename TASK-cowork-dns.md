# Задача для Claude Cowork: перенести домен jondey.kz на Vercel

**Цель:** чтобы сайт `https://jondey.kz` открывался с Vercel (проект `jondey-site`).
Сейчас домен ведёт на старый хостинг ps.kz. Нужно поменять 3 DNS-записи и добавить
домен в Vercel. **Это правки DNS-записей, НЕ удаление файлов/сайтов.**

## Текущее состояние DNS (проверено)
- `jondey.kz` → A **194.39.65.25** (старый хостинг) → надо **76.76.21.21** (Vercel)
- `jondey.kz` → AAAA **2a00:5da0:1000::167** → надо **удалить эту запись**
- `www.jondey.kz` → CNAME на `jondey.kz` → надо **cname.vercel-dns.com**
- MX → `mail.jondey.kz` → **НЕ ТРОГАТЬ** (почта)

---

## ЧАСТЬ A — Vercel (добавить домен)
1. Открыть **vercel.com**, войти в аккаунт владельца.
2. Открыть проект **jondey-site** → вкладка **Settings** → раздел **Domains**.
3. В поле ввода написать `jondey.kz` → нажать **Add**.
4. Ещё раз: написать `www.jondey.kz` → **Add**.
5. Vercel покажет требуемые DNS-записи. Запомнить значение для A
   (по умолчанию **76.76.21.21**) и для www CNAME (**cname.vercel-dns.com**).
   ⚠️ Если Vercel показывает другое значение A — использовать ЕГО.

## ЧАСТЬ B — DNS в панели (ps.kz / Plesk)
Открыть управление DNS зоны `jondey.kz`. Два возможных места:
- **Plesk:** Сайты и домены → `jondey.kz` → **Параметры DNS** (DNS Settings).
- Или **ps.kz личный кабинет:** Домены → `jondey.kz` → Управление DNS / DNS-зона.
  (Зона там, где записи совпадают с текущими: A 194.39.65.25.)

Сделать **ровно 3 правки**:

1. **A-запись** `@` (или `jondey.kz`):
   изменить значение `194.39.65.25` → **`76.76.21.21`**. Сохранить.

2. **AAAA-запись** `@` (значение `2a00:5da0:1000::167`):
   **удалить** эту запись целиком. (Это IPv6 старого хостинга; Vercel её не использует.)

3. **CNAME** `www`:
   изменить значение `jondey.kz` → **`cname.vercel-dns.com.`** (можно с точкой на конце).
   Сохранить.

Нажать **Применить / Update** если есть такая кнопка.

## СТРОГО НЕ ТРОГАТЬ
- ❌ MX-записи и всё почтовое (`mail.jondey.kz`, autodiscover, SPF/DKIM/TXT).
- ❌ Сам домен/подписку в Plesk (не удалять домен — на нём зона и почта).
- ❌ Записи NS (`ns1/ns2/ns3.ps.kz`).
- ❌ Другие сайты на хостинге (httpdocs, winttechnology.com).

## ЧАСТЬ C — финал в Vercel (после применения DNS)
1. В Vercel → Settings → Domains дождаться у `jondey.kz` статуса **Valid Configuration**
   (зелёная галочка). Может занять от 10 минут до пары часов.
2. Сделать `jondey.kz` основным доменом (Production), а `www` — редирект на него
   (Vercel предложит кнопкой «Redirect to jondey.kz»).
3. Settings → **Environment Variables**: задать
   `NEXT_PUBLIC_SITE_URL` = `https://jondey.kz` → **Save** → нажать **Redeploy**
   последнего деплоя (Deployments → ⋯ → Redeploy).

## Проверка (готово, если):
- `https://jondey.kz` открывает сайт Jondey (не заглушку ps.kz).
- В адресной строке замок (SSL выпущен Vercel).
